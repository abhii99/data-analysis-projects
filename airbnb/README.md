# Amsterdam-Airbnb-Analysis

Exploratory data analysis of Airbnb listings in Amsterdam using Python, focusing on pricing patterns, geographic trends, and host behavior.  
Dataset snapshot: December 6th, 2018 (~20K+ records)      
Full dataset details: https://www.kaggle.com/datasets/erikbruin/airbnb-amsterdam/data

## Key Insights 

- Central neighborhoods (e.g., Centrum-West) have the highest average prices
- Entire homes command significantly higher prices than private/shared rooms
- Host verification shows measurable differences in review patterns
