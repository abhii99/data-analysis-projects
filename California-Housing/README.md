# California Housing Price Analysis

Predictive modeling and exploratory data analysis on California housing data using regression techniques.

## Overview
This project analyzes housing data from the 1990 California census to identify key factors influencing median house prices and build a predictive regression model.

The analysis includes data exploration, feature selection, statistical validation, and model evaluation.

## Key Insights
- Median income is the strongest predictor of housing prices
- Geographic location (latitude & longitude) significantly improves model performance
- Homes near coastal/bay areas tend to have higher values
- Adding spatial features improved R² from ~0.50 to ~0.60 and reduced RMSE by ~8%

## Methods & Tools
- Python (Pandas, NumPy, Seaborn, Matplotlib)
- Linear Regression
- Ordinary Least Squares (OLS)
- Exploratory Data Analysis (EDA)
- Feature Engineering
